Toast Нотификације

Овако ће изгледати аутоматске поруке након submit-а форми на твом сајту. Кликни на дугмад испод да видиш примере:

Карактеристике:

  • Аутоматско појављивање - Toast се појави одмах након submit-а
  • Смут анимације - Slide-in и fade-out ефекти
  • Ауто-затварање - Нестаје након 5 секунди
  • Респонсивно - Прилагођава се мобилном екрану
  • Без alert() - Модерна UI нотификација
// Пример коришћења:

// Успешна резервација
EmailService.showMessage(
  'Резервација је послата! Проверите ваш email за потврду.',
  true // success = зелена боја
);

// Грешка
EmailService.showMessage(
  'Дошло је до грешке. Покушајте поново.',
  false // error = црвена боја
);
